# Documentação

Toda a documentação do projeto está organizada nos arquivos abaixo:

- [architecture.md](architecture.md): visão técnica da arquitetura, componentes e instruções para recriar o diagrama no draw.io.
- [provisioning.md](provisioning.md): fluxos de provisionamento (manual e via GitHub Actions), pré‑requisitos e estrutura do repositório.
- [BACKEND_AND_STATE.md](BACKEND_AND_STATE.md): guia completo para configurar e manter o backend remoto (bucket GCS + locking).
